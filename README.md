# HanceWeb 2.0 (個人網站專案)

## 簡介

---

利用 React 和 ExpressJS 開發的個人網站，主要展示個人作品集與介紹關於自己的技能、學歷、簡歷等。也包括管理後台的功能，可以直接在自製的後台介面修改資料。
